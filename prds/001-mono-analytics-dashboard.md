## 1. Executive Summary

### Problem
Most analytics dashboards look generated by AI—gradient overload, excessive shadows, and decorative noise that distracts from data. Teams need clean, focused dashboards that present information clearly without visual fluff.

### Solution
Build a self-contained, monospace analytics dashboard framework inspired by MonkeyType's utilitarian aesthetic. The framework will be:
- **Zero-dependency**: Pure HTML/CSS/JS, no external libraries
- **Reusable**: Component-based architecture for any data type
- **Focused**: Data-first design with minimal visual noise
- **Monospace**: Single font family throughout (JetBrains Mono)
- **Configurable**: Theme and data agnostic

## 6. Visual Design System

### Design Principles (MonkeyType-inspired)

1. **Content over chrome**: No decorative elements
2. **Single accent**: One color draws attention
3. **Borders over shadows**: 1px solid lines define structure
4. **Monospace everywhere**: JetBrains Mono for all text
5. **Whitespace is structure**: Generous padding creates hierarchy
6. **Subtle interactions**: Color changes, not transforms

### Color Palette (Default: Dark)

| Token | Hex | Usage |
|-------|-----|-------|
| `--bg` | `#111111` | Page background |
| `--surface` | `#1a1a1a` | Card/panel backgrounds |
| `--surface-hover` | `#222222` | Hover states |
| `--border` | `#333333` | Borders and dividers |
| `--border-focus` | `#444444` | Focus/active borders |
| `--text` | `#d1d1d1` | Primary text |
| `--text-muted` | `#666666` | Secondary text |
| `--accent` | `#e2b714` | Accent color (MonkeyType yellow) |
| `--accent-hover` | `#f0c000` | Accent hover |
| `--error` | `#ca4754` | Error states |
| `--success` | `#7cb97a` | Success states |

### Typography
- **Font**: JetBrains Mono (monospace)
- **Weights**: 400 (regular), 700 (bold)
- **Line height**: 1.5
- **Hierarchy**:
  - Title: 2rem, weight 700
  - Section headers: 1rem, weight 700, uppercase, letter-spacing 0.1em
  - Values: 1.5rem, weight 700
  - Labels: 0.75rem, weight 400, uppercase, letter-spacing 0.05em
  - Body: 0.875rem, weight 400

### Spacing System
- **Base unit**: 4px
- **Page padding**: 24px
- **Section gap**: 32px
- **Component gap**: 16px
- **Inner padding**: 16px

### Borders
- **Default**: 1px solid var(--border)
- **Radius**: 4px (subtle rounding)
- **Focus**: 1px solid var(--accent)

### Effects
- **None**: No shadows, no glows, no gradients
- **Transitions**: `color 0.1s ease, background-color 0.1s ease`
- **Focus outline**: 2px solid var(--accent), offset 2px

## 9. Appendix

### Reference: MonkeyType Design Elements
- Solid dark background (#111)
- Single yellow accent (#e2b714)
- 1px borders (#333)
- No shadows, no gradients
- Monospace throughout
- Uppercase labels with letter-spacing
- Generous whitespace
- Subtle 4px border-radius
- Focus: color change, not motion

### Data Schema
```typescript
interface DashboardData {
  stats: Array<{
    icon: string;
    label: string;
    value: number | string;
    change?: string;
    sub?: string;
  }>;
  items: Array<{
    id: string;
    name: string;
    values: Record<string, number>;
    total: number;
  }>;
  metadata?: {
    title: string;
    subtitle: string;
    source: string;
  };
}
```
